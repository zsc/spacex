## 10.1 传统航天材料的局限性

### 10.1.1 铝合金时代

自航天工业诞生以来，铝合金一直是火箭结构的主流材料。其优势明显：

```
铝合金特性矩阵
┌────────────────────────────────────────┐
│ 优势                   劣势             │
├────────────────────────────────────────┤
│ • 密度低 (2.7 g/cm³)  • 低温脆化       │
│ • 成熟工艺            • 高温软化       │
│ • 良好加工性          • 疲劳裂纹       │
│ • 供应链完善          • 焊接难度大     │
└────────────────────────────────────────┘
```

**案例分析：Falcon 9的铝锂合金**

Falcon 9采用2195铝锂合金，这是航天工业的黄金标准：
- 密度：2.63 g/cm³（比纯铝低4%）
- 极限抗拉强度：560 MPa
- 工作温度范围：-253°C 至 150°C
- 成本：~$30/kg

然而，即使是最先进的铝锂合金，在极端温度环境下也面临挑战：


### 10.1.2 碳纤维复合材料的诱惑

2016年，当马斯克首次公布ITS（Interplanetary Transport System）时，选择了碳纤维复合材料作为主体结构：

**理论优势：**
- 极高比强度：1600 MPa/1.6 g/cm³ = 1000 kN·m/kg
- 可设计性：纤维方向可优化应力分布
- 零热膨胀系数：尺寸稳定性极佳
- 疲劳性能优异：循环载荷下性能稳定

**SpaceX的碳纤维实践（2016-2018）：**

```

         • 成本：$200,000/吨
```

## 10.2 碳纤维梦想的破灭

### 10.2.1 制造地狱

2017-2018年间，SpaceX在洛杉矶港建造了世界最大的碳纤维复合材料制造设施。然而，实际制造过程暴露了严重问题：

**1. 低温燃料相容性危机**

```
低温性能对比 (-196°C液氮环境)
                    微裂纹密度(/cm²)
碳纤维/环氧树脂  ████████████████ 1600
碳纤维/聚酰亚胺  ████████ 800
铝合金           ██ 200  
304L不锈钢       ▌ 50
```

液氧和液甲烷的极低温度导致：
- 基体树脂脆化开裂
- 纤维/基体界面分离
- 微裂纹扩展导致燃料渗漏

**2. 规模化制造噩梦**

```
制造流程复杂度对比
碳纤维复合材料制造流程：
[铺层] → [真空袋] → [固化72h] → [脱模] → [机加工] → [无损检测]
  2天      4小时       3天        1天       2天         1天
                    总计：~9天/段

不锈钢制造流程：
[切割] → [卷板] → [焊接] → [检测]
 2小时    4小时    8小时    2小时
                    总计：~2天/段
```

**3. 质量控制困境**

碳纤维结构的无损检测极其复杂：
- 超声波检测：需要逐点扫描，耗时巨大
- X射线检测：对分层缺陷不敏感
- 热成像检测：只能发现表层缺陷

### 10.2.2 成本失控

## 10.11 结论：第一性原理的胜利

### 10.11.1 核心启示

不锈钢的选择完美诠释了第一性原理思维：

1. **质疑一切假设**：为什么火箭必须用"先进"材料？
2. **回归物理本质**：什么材料真正满足任务需求？
3. **系统性思考**：材料选择如何影响整个系统？
4. **快速迭代验证**：用实验而非理论验证假设
